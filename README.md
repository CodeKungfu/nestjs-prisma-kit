# 初始化 primsma

npx prisma generate

npm run db:gen

# 部署

1，clone 代码 

2，yarn instlal

3， 运行sql （sql/sql.sql）

4, 新建.env
配置好.env 参考.env.local

5, npx prisma db pull (yarn db:pull)

6, yarn db:gen

7, 启动代码: yarn dev
